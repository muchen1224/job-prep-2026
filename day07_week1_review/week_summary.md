# 第一周学习总结

## 一、本周学习目标

本周主要完成 C 语言基础复习、Python 基础入门、Linux/Git 基础操作，以及 AI Agent 的概念认知，为后续嵌入式、IoT 和智能运维方向学习打基础。

## 二、本周完成内容

### 1. C 语言基础
完成了 1 到 100 求和、九九乘法表、数组最大值等练习，复习了变量、循环、判断和数组。

### 2. C 语言进阶基础
学习了指针、结构体、字符串和位运算，初步理解了这些内容在嵌入式开发中的作用。

### 3. Python 基础
学习了 Python 的变量、列表、字典、函数、文件读写和 JSON，完成了设备信息管理相关脚本。

### 4. Linux 和 Git
掌握了常见终端命令，能够使用 Git 完成 add、commit、push，并将学习代码上传到 GitHub。

### 5. Agent 基础
初步理解了 Agent 不是普通聊天机器人，而是由模型、工具、状态和执行逻辑组成的系统。

## 三、本周完成的代码

- C 语言基础练习
- 指针和结构体练习
- Python JSON 文件读写练习
- 设备信息管理小工具

## 四、本周遇到的问题

- C 语言中 `*` 和 `&` 容易混淆
- Git 初次上传流程不熟悉
- Python 文件读写和 JSON 还需要继续练习
- Agent 概念还停留在理解阶段，后续需要写实际 demo

## 五、本周最大的收获

本周最大的收获是初步建立了持续学习和代码留痕的习惯。通过 GitHub 管理每日代码，可以让学习过程更加清晰，也为后续找工作积累项目材料。

## 六、下周学习计划

下周准备进入 STM32 基础学习，重点学习 GPIO、串口 UART、定时器和 ADC，同时继续用 Python 做简单自动化脚本，为后续设备监控项目做准备。
